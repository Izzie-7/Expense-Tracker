Expense Tracker Web Application

Project Overview
This is a web-based Expense Tracker application that allows users to:
- Add income and expense transactions
- View transaction history
- Filter transactions by category and description
- Switch between light and dark modes
- Convert amounts between Naira and Dollars

Features
- Add and delete transactions
- Track total income, expenses, and current balance
- Search and filter transactions
- Responsive design
- Dark/Light mode Button
- An Event confirming if users want to delete a transaction

How to Use
1. Clone or download the project folder
2. Open `index.html` in a web browser
3. Click "Add Transaction" to record new income or expenses
4. Use the search and filter options to find specific transactions
5. Switch between light and dark modes using the sun/moon icon

Steps to Load the Project
1. Download the project folder
2. Ensure all files are in their respective folders (HTML, CSS, JavaScript, Images)
3. Open `index.html` in any web browser (Chrome, Firefox, Safari or MS Edge)
4. No additional software or dependencies are required

Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage for data persistence

And Note:
- All transactions are stored in the browser's local storage
- Conversion rate: 1 USD = 1,550 NGN (approximate)
